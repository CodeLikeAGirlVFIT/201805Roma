A bit about the internet

URL

URL sta per Uniform Resource Locator. Contiene 3 parti:

  • the protocol: http,https,ftp
  • the host: e.g. Facebook
  • the path: e.g. /news/local

Client-side

Client-side gira nel tuo web browser.

  1. HTML
  2. CSS
  3. Javascript

Server-side

Server-side girano sulle machine server (la parte hardware)
